# Code of Conduct

## Purpose

This repository is a private, closed-source project. The code of conduct applies
to all maintainers, contributors, reviewers, contractors, institutional
reviewers, and other authorized users with access to the repository.

## Expected Behavior

Participants are expected to:

- Use repository access only for the authorized purpose.
- Treat source code, documents, credentials, issue discussions, pull requests,
  and review findings as confidential.
- Communicate professionally and respectfully.
- Provide technical feedback that is specific, evidence-based, and relevant to
  the project.
- Protect local clones, exported files, screenshots, credentials, and access
  tokens from unauthorized disclosure.
- Report suspected security issues, leaked secrets, unauthorized access, or
  license violations promptly to the repository owner.

## Unacceptable Behavior

The following behavior is not permitted:

- Sharing, publishing, copying, redistributing, or transferring repository
  contents without written authorization.
- Using repository materials outside the authorized review or collaboration
  purpose.
- Harassment, threats, discriminatory language, personal attacks, or abusive
  conduct.
- Attempting to bypass access controls, extract secrets, abuse test credentials,
  or access systems outside the approved scope.
- Posting sensitive review findings publicly before the repository owner has had
  a reasonable opportunity to review and respond.
- Removing copyright, license, proprietary, confidentiality, or attribution
  notices.

## Security and Confidentiality Reports

Report security issues, suspected leaks, or unauthorized use privately to the
repository owner. Do not disclose sensitive findings publicly unless the
repository owner has given written permission or a legal obligation requires it.

## Enforcement

The repository owner may remove access, revoke permissions, delete comments,
close issues, reject contributions, or take further action when this code of
conduct, the [license](LICENSE), or the [copyright notice](COPYRIGHT.md) is
violated.
