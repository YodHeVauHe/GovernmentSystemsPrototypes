# Contributing

This is a private, closed-source repository. Contributions are accepted only
from explicitly authorized collaborators.

## Contribution Rules

- Do not copy repository code, documents, designs, API specifications, or assets
  into another project without written permission.
- Do not submit third-party code unless its license permits private proprietary
  use and redistribution inside this repository.
- Do not add real citizen data, production credentials, live government secrets,
  or uncontrolled personal data.
- Keep review comments, issue discussions, pull requests, screenshots, and local
  clones confidential.
- Follow the [code of conduct](CODE_OF_CONDUCT.md), [security policy](SECURITY.md),
  [copyright notice](COPYRIGHT.md), and [proprietary license](LICENSE).

## Review Access

Government, institutional, procurement, security, or technical-review access is
for evaluation only. It does not grant the right to reuse, deploy, publish,
modify, redistribute, or commercialize the repository or its contents.
